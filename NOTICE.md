# Notice

This repository is a public case study for **Briefnexa**, a privacy-focused AI meeting assistant MVP.

## Public showcase only

This repository is intended to communicate the product concept, MVP scope, architecture decisions, privacy approach, and selected demo screens. It is not the production codebase.

## Source code is not included

The source code is intentionally private because the project is actively being developed as a product.

This repository does not include:

- frontend source files
- backend source code
- database migrations
- API keys
- credentials
- `.env` files
- logs
- signed URLs
- private transcripts
- real user data
- datasets
- provider payloads
- storage keys

## Screenshots

Screenshots in this repository should use demo data only. They should not contain private meeting data, personal information, real users, private emails, phone numbers, credentials, transcripts, signed URLs, or sensitive content.
